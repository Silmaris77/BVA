## SLAJD 6: KLIENCI - Lifecycle

```
┌─────────────────────────────────────────────────┐
│  👥 CYKL ŻYCIA KLIENTA                          │
│                                                 │
│  PROSPECT 🎯                                    │
│  ├─ Cold call (trudna rozmowa)                 │
│  ├─ 1-3 wizyty potrzebne                       │
│  └─ Cel: Podpisać kontrakt                     │
│      ↓ [Kontrakt] ✅                            │
│                                                 │
│  ACTIVE 😊                                      │
│  ├─ Check-in co 7-14 dni                       │
│  ├─ Reputacja: -100 do +100                    │
│  ├─ Market share: 20-40%                       │
│  └─ Możliwości: Cross-sell, Up-sell            │
│      ↓ [Zaniedbanie / Rep <-50] ⚠️             │
│                                                 │
│  LOST 😢                                        │
│  ├─ Win-back: trudna rozmowa                   │
│  ├─ Wymaga: przeprosiny + oferta ekstra        │
│  └─ Max 3 próby odzyskania                     │
│      ↓ [Win-back sukces] 🎉                    │
│                                                 │
│  ACTIVE (rep=0, rebuild trust)                 │
└─────────────────────────────────────────────────┘
```

**Przykład flow:**
1. Prospect "Sklep ABC" → 2 wizyty (4⭐, 5⭐) → Kontrakt
2. Active przez 45 dni → rep 70 → Cross-sell sukces
3. Zaniedbanie (14 dni brak wizyty) → rep -30 → LOST
4. Win-back z Gratis → 5⭐ → odzyskany, rep=0

## SLAJD 7: REPUTACJA - Dwupoziomowy System

```
┌─────────────────────────────────────────────────┐
│  ⭐ SYSTEM REPUTACJI                            │
│                                                 │
│  POZIOM 1: REPUTACJA U KLIENTA                 │
│  ────────────────────────────────────────────  │
│  Każdy ACTIVE/LOST klient: -100 do +100       │
│                                                 │
│   -100        -50         0         50    100  │
│    ├───────────┼──────────┼──────────┼─────┤  │
│    Wściekły  Niezad.  Neutral  Happy  VIP     │
│                                                 │
│  Wpływ:                                        │
│  • Rozmowy 5⭐ → +10 rep                       │
│  • Spóźniona wizyta → -5 rep                   │
│  • Problem rozwiązany → +10 rep                │
│  • Rep <-50 → LOST (utrata klienta)           │
│                                                 │
│  ───────────────────────────────────────────   │
│                                                 │
│  POZIOM 2: REPUTACJA OGÓLNA                    │
│  ────────────────────────────────────────────  │
│  Weighted average wszystkich klientów          │
│                                                 │
│  Formula:                                      │
│  ┌─────────────────────────────────────────┐  │
│  │ (Σ ACTIVE × PLN) × 0.8                  │  │
│  │ + (avg LOST) × 0.2                      │  │
│  └─────────────────────────────────────────┘  │
│                                                 │
│  Przykład: 54/100 = "Solid Rep" ✅             │
│                                                 │
│  Wpływ na grę:                                 │
│  • ≥90: VIP prospects (word of mouth)         │
│  • ≥60: Wymagane do awansu Level 2            │
│  • <40: Email od szefa (ostrzeżenie)          │
└─────────────────────────────────────────────────┘
```

**Insight:**
Portfolio balancing = strategia!  
1 VIP (5k PLN, rep 90) ≠ 5 małych (1k, rep 60)
