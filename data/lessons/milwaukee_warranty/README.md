# Milwaukee - Warunki Gwarancji

## Cel folderu
Materiały do stworzenia lekcji o warunkach gwarancji dla użytkowników Milwaukee.

## Co tutaj umieścić?
- **materialy_zrodlowe/** - Dokumenty PDF, prezentacje, instrukcje gwarancyjne
- **notatki.md** - Notatki i struktura lekcji
- **media/** - Zdjęcia, schematy, infografiki
- **quiz_pytania.txt** - Pytania do quizu końcowego

## Format docelowy
Lekcja zostanie utworzona jako plik JSON w `data/lessons/MILWAUKEE_Warunki_Gwarancji.json`

## Struktura lekcji (planowana)
1. **Intro** - Wprowadzenie do gwarancji Milwaukee
2. **Sekcja 1** - Podstawowe warunki gwarancji
3. **Sekcja 2** - Procedury reklamacyjne
4. **Sekcja 3** - Najczęstsze pytania i błędy
5. **Quiz** - Test wiedzy
6. **Podsumowanie** - Kluczowe wnioski

## Tagi zasobów
Po utworzeniu lekcji pamiętaj dodać w `config/resource_tags.json`:
```json
"MILWAUKEE_Warunki_Gwarancji": ["Milwaukee"]
```
